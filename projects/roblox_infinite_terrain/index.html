Roblox Infinite Terrain

I came up with the idea to make a horror game in roblox where you walk around in an infinite glass maze and use items you find to try to escape. There were a lot of bugs so i decided i wanted to try again from scratch. Later i came back to the idea but instead of a maze i wanted to use perlin noise to generate terrain. This is the "finished" product, you can go as far as you want in any direction, it also has a Depth Of Field effect and a fog effect that whas new at the time. It also saves how far you have traveled and displays it in a global leaderboard.

Link to the Roblox game. You need a free account to play it, but you only have to install Roblox, not any of the games.
